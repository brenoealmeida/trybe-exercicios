Tudo sobre a Sequoia-gigante

Descrição:

A sequoia-gigante é a maior arvore do mundo em termos de volume. Ela atinge em media de 85-115m de altura e 8-12m de diametro. A sequoia-gigante mais velha conhecida possui 4.650 anosde idade e se encontra no Parque Nacional Sequoiana California.
